# Instructions pour télécharger les fichiers audio des nouveaux animaux

## Liens directs pour télécharger les fichiers audio :

### 1. Chèvre (chevre.wav)
- **Pixabay**: https://pixabay.com/fr/sound-effects/search/goat/
- **Freesound**: https://freesound.org/search/?q=goat+baa
- **Recherche**: "goat baa sound", "chèvre bêlement"

### 2. Souris (souris.wav)
- **Pixabay**: https://pixabay.com/fr/sound-effects/search/mouse/
- **Freesound**: https://freesound.org/search/?q=mouse+squeak
- **Recherche**: "mouse squeak sound", "souris couic", "mouse eating sound"

### 3. Poule (poule.wav)
- **Pixabay**: https://pixabay.com/fr/sound-effects/search/chicken/
- **Freesound**: https://freesound.org/search/?q=chicken+cluck
- **Recherche**: "chicken cluck sound", "poule cot cot"

### 4. Lapin (lapin.wav)
- **Pixabay**: https://pixabay.com/fr/sound-effects/search/rabbit/
- **Freesound**: https://freesound.org/search/?q=rabbit+eating
- **Recherche**: "rabbit eating sound", "lapin grignotage", "rabbit chewing"

### 5. Âne (ane.wav)
- **Pixabay**: https://pixabay.com/fr/sound-effects/search/donkey/
- **Freesound**: https://freesound.org/search/?q=donkey+hee-haw
- **Recherche**: "donkey hee-haw sound", "âne hi-han"

### 6. Dinde (dinde.wav)
- **Pixabay**: https://pixabay.com/fr/sound-effects/search/turkey/
- **Freesound**: https://freesound.org/search/?q=turkey+gobble
- **Recherche**: "turkey gobble sound", "dinde glouglou"

## Instructions de téléchargement :

1. Visitez l'un des sites ci-dessus
2. Recherchez le son de l'animal
3. Téléchargez le fichier (format WAV ou MP3 recommandé)
4. Renommez-le selon le nom attendu (ex: `chevre.wav`)
5. Placez-le dans le dossier `apprendre-autrement/public/sounds/`
6. Redémarrez le container Docker pour que les fichiers soient pris en compte

## Format recommandé :
- **Format**: WAV ou MP3
- **Qualité**: 44.1 kHz, 128 kbps minimum
- **Durée**: 1-3 secondes
- **Taille**: < 500 KB par fichier

## Sources Recommandées pour Télécharger les Fichiers

1. **Pixabay** : https://pixabay.com/fr/sound-effects/search/sons%20d%27animaux/
   - Recherche par nom d'animal en français ou anglais
   - Fichiers libres de droits

2. **Freesound** : https://freesound.org/browse/tags/animals/
   - Grande base de données de sons
   - Nécessite un compte gratuit pour télécharger

3. **Zapsplat** : https://www.zapsplat.com/sound-effect-categories/animal-sounds/
   - Sons de qualité professionnelle
   - Nécessite un compte gratuit

4. **La Sonothèque** : https://lasonotheque.org/
   - Banque de sons gratuits en français
   - Recherche par catégorie "Animaux"

## Instructions

1. Téléchargez les fichiers audio au format MP3
2. Renommez-les selon la liste ci-dessus (ex: `dog.mp3`, `cat.mp3`, etc.)
3. Placez-les dans le dossier `public/sounds/`
4. Les fichiers doivent être de bonne qualité (44.1 kHz, 128 kbps minimum)
5. Durée recommandée : 1-3 secondes par fichier

## Format des Fichiers

- **Format** : MP3 (recommandé) ou WAV
- **Qualité** : 44.1 kHz, 128 kbps minimum
- **Durée** : 1-3 secondes par fichier
- **Taille** : Essayer de garder chaque fichier sous 500 KB

## Note

Si les fichiers audio ne sont pas disponibles, l'application utilisera automatiquement la synthèse vocale comme solution de repli.

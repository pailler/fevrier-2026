# Sons d'Animaux

Ce dossier contient les fichiers audio des cris d'animaux pour l'activité "Les Cris d'Animaux".

## Fichiers requis

Pour que l'activité fonctionne avec de vrais sons, placez les fichiers audio suivants dans ce dossier :

- `dog.mp3` - Cri de chien
- `cat.mp3` - Cri de chat
- `rooster.mp3` - Cri de coq
- `cow.mp3` - Cri de vache
- `sheep.mp3` - Cri de mouton
- `pig.mp3` - Cri de cochon
- `horse.mp3` - Cri de cheval
- `duck.mp3` - Cri de canard
- `lion.mp3` - Cri de lion
- `elephant.mp3` - Cri d'éléphant
- `monkey.mp3` - Cri de singe
- `owl.mp3` - Cri de hibou
- `wolf.mp3` - Cri de loup
- `bear.mp3` - Cri d'ours
- `frog.mp3` - Cri de grenouille
- `bee.mp3` - Bruit d'abeille

## Sources recommandées

Vous pouvez télécharger des fichiers audio libres de droits depuis :

- **Pixabay** : https://pixabay.com/fr/sound-effects/search/sons%20d%27animaux/
- **Freesound** : https://freesound.org/browse/tags/animals/
- **Zapsplat** : https://www.zapsplat.com/sound-effect-categories/animal-sounds/

## Format

- Format recommandé : MP3
- Qualité : 44.1 kHz, 128 kbps minimum
- Durée : 1-3 secondes par fichier

## Note

Si les fichiers audio ne sont pas disponibles, l'application utilisera automatiquement la synthèse vocale comme solution de repli.

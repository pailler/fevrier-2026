# Guide de téléchargement des bruits de la ville

## Fichiers audio nécessaires

Pour l'activité "Les Bruits de la Ville", les fichiers audio suivants doivent être téléchargés dans le dossier `public/sounds/` :

1. **fire-truck.mp3** - Bruit de camion de pompiers
2. **garbage-truck.mp3** - Bruit de camion poubelle
3. **police-siren.mp3** - Bruit de sirène de police
4. **ambulance.mp3** - Bruit d'ambulance
5. **motorcycle.mp3** - Bruit de moto
6. **car.mp3** - Bruit de voiture

## Sources recommandées

- **Freesound.org** : https://freesound.org/
  - Rechercher "fire truck sound", "garbage truck", "police siren", "ambulance siren", "motorcycle sound", "car engine"
  - Filtrer par licence CC0 ou CC BY (libre d'utilisation)

- **Zapsplat** : https://www.zapsplat.com/
  - Rechercher les mêmes termes
  - Nécessite un compte gratuit

- **YouTube Audio Library** : https://studio.youtube.com/channel/UC.../music
  - Rechercher dans la bibliothèque audio gratuite

## Format recommandé

- Format : MP3
- Qualité : 128-192 kbps
- Durée : 2-5 secondes par son
- Volume : Normalisé pour éviter les différences trop importantes

## Instructions

1. Téléchargez les fichiers audio depuis une source libre de droits
2. Renommez-les exactement comme indiqué ci-dessus
3. Placez-les dans le dossier `apprendre-autrement/public/sounds/`
4. Testez l'activité pour vérifier que les sons fonctionnent correctement

## Note

Si les fichiers audio ne sont pas disponibles, l'application utilisera la synthèse vocale comme solution de secours.

# Nouveaux animaux à ajouter - Fichiers audio requis

## Animaux à ajouter avec leurs cris ou sons de grignotage/manger :

1. **Chèvre** (Goat)
   - Fichier : `chevre.wav` ou `chevre.mp3`
   - Cri : Bêêê / Maa maa
   - Ou son de grignotage si disponible

2. **Souris** (Mouse)
   - Fichier : `souris.wav` ou `souris.mp3`
   - Cri : Couic couic / Squeak
   - Ou son de grignotage si disponible

3. **Poule** (Hen)
   - Fichier : `poule.wav` ou `poule.mp3`
   - Cri : Cot cot codet / Cluck cluck

4. **Lapin** (Rabbit)
   - Fichier : `lapin.wav` ou `lapin.mp3`
   - Cri : (peu de son)
   - Ou son de grignotage si disponible

5. **Âne** (Donkey)
   - Fichier : `ane.wav` ou `ane.mp3`
   - Cri : Hi-han / Hee-haw

6. **Dinde** (Turkey)
   - Fichier : `dinde.wav` ou `dinde.mp3`
   - Cri : Glouglou / Gobble gobble

## Sources pour télécharger les fichiers :

- Pixabay : https://pixabay.com/fr/sound-effects/search/sons%20d%27animaux/
- Freesound : https://freesound.org/browse/tags/animals/
- Zapsplat : https://www.zapsplat.com/sound-effect-categories/animal-sounds/

## Instructions :

1. Téléchargez les fichiers audio
2. Placez-les dans le dossier `public/sounds/`
3. Les animaux apparaîtront automatiquement dans l'activité une fois les fichiers ajoutés

# Optimisations SEO par application – IA Home

Ce document récapitule les optimisations proposées pour améliorer le classement Google de chaque page application (card).

## Principes appliqués

- **Title** : 50–60 caractères, mot-clé principal en début, marque « | IA Home » en fin.
- **Description** : 155–160 caractères, bénéfice + CTA ou chiffre, mots-clés longue traîne.
- **Keywords** : requêtes à fort volume + variantes FR + longue traîne + nom de l’outil/marque.
- **OpenGraph / Twitter** : titres et descriptions dédiés (éventuellement raccourcis), alt d’images descriptifs.

## Mise en œuvre

Les changements sont appliqués dans les fichiers `src/app/card/[app]/layout.tsx` de chaque application :  
`metadata.title`, `metadata.description`, `metadata.keywords`, et blocs `openGraph` / `twitter` (titres, descriptions, alt des images).
